# demsg 命令



 demsg命令用于显示开机信息，内核会将开机信息存储在系统缓冲区（ring buffer）中，开机后可用dmesg命令查看，也可以在/var/log/目录中查看dmesg文件。用法如下：

**命令参数**

| **选项** | **含义**                            |
| -------- | ----------------------------------- |
| -c       | 显示开机信息后，清除ring buffer信息 |
| -s       | 设置缓冲区大小，默认设置为8192      |
| -n       | 设置记录信息的层级                  |

  内核缓冲区的信息对于诊断系统非常有用。在查看时可以配合grep使用。
