安装工具：

```bash
$ yum install bridge-utils
```

查看网桥列表：

```bash
$ brctl show
```

查看网络命名空间：

```
$ ip netns
$ ls /var/run/netns
```

在某个命名空间中查看网卡：

```bash
$ ip netns exec qdhcp-a5c62f47-b1c1-4a8e-a20d-81371017843a ip a
```

查看网关：

```
$ netstat -rn
```
