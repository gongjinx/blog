# 2016年美国总统选举辩论数据检索实战

数据来源：http://dataju.cn/Dataju/web/datasetInstanceDetail/268

下载下来之后是一个 CSV 数据，通过 kibana 的 Data Visualizer  功能导入到 Elasticsearch 中，之后就可以查询了。

一个简单的 bool 查询：

````http
GET /race-for-president-2016/_search
{
  "query": {
    "bool": {
      "must": [
        { "match": { "Text": "New York"}}
      ],
      "filter": [
        {"term": { "Speaker": "Holt" }}
      ]
    }
  }
}
````
